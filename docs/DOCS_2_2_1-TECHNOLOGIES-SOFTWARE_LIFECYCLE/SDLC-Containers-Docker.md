# SDLC-Containers-Docker

## Ресурсы для изучения Docker

* [docker/awesome-compose: Awesome Docker Compose samples (github.com)](https://github.com/docker/awesome-compose)

## Вспомогательные сторонние инструменты для Docker

* [open-source-labs/Docketeer: An easy-to-use GUI for Docker that allows developers to manage Docker containers and monitor crucial metrics. (github.com)](https://github.com/open-source-labs/Docketeer)

* [12 друзей Docker-а – опенсорсные инструменты в помощь разработке (proglib.io)](https://proglib.io/p/12-docker-tools)

* [docker/docker-bench-security: The Docker Bench for Security is a script that checks for dozens of common best-practices around deploying Docker containers in production. (github.com)](https://github.com/docker/docker-bench-security)

* [jesseduffield/lazydocker: The lazier way to manage everything docker (github.com)](https://github.com/jesseduffield/lazydocker)

* [cyberark/KubiScan: A tool to scan Kubernetes cluster for risky permissions (github.com)](https://github.com/cyberark/KubiScan)

* [wagoodman/dive: A tool for exploring each layer in a docker image (github.com)](https://github.com/wagoodman/dive)

* [bcicen/ctop: Top-like interface for container metrics (github.com)](https://github.com/bcicen/ctop)

* [aquasecurity/trivy: Find vulnerabilities, misconfigurations, secrets, SBOM in containers, Kubernetes, code repositories, clouds and more (github.com)](https://github.com/aquasecurity/trivy)

* [whalebrew/whalebrew: Homebrew, but with Docker images (github.com)](https://github.com/whalebrew/whalebrew)

* [slimtoolkit/slim: Slim(toolkit): Don't change anything in your container image and minify it by up to 30x (and for compiled languages even more) making it secure too! (free and open source) (github.com)](https://github.com/slimtoolkit/slim)

* [hadolint/hadolint: Dockerfile linter, validate inline bash, written in Haskell (github.com)](https://github.com/hadolint/hadolint)

* [EugenMayer/docker-sync: Run your application at full speed while syncing your code for development, finally empowering you to utilize docker for development under OSX/Windows/\*Linux (github.com)](https://github.com/eugenmayer/docker-sync)
